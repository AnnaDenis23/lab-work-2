## 📋 Описание
Windows Forms приложение на C#, созданное в рамках лабораторной работы по технологиям программирования. Проект демонстрирует основы разработки desktop-приложений с использованием .NET Framework.

## 🚀Запуск

- 💻Откройте `ЛАБА2.1.csproj` в Visual Studio 2019/2022

- ⚙️ .NET Framework 4.7.2 или выше

- Нажмите F5 для запуска
